准备环境

```
curl -sLo go.tar.gz https://go.dev/dl/$(curl -sL https://golang.org/VERSION?m=text|head -1).linux-arm64.tar.gz
rm -rf /usr/local/go
tar -C /usr/local/ -xzf go.tar.gz
rm go.tar.gz
echo -e "export PATH=$PATH:/usr/local/go/bin" > /etc/profile.d/go.sh
source /etc/profile.d/go.sh
go version
```

```
apt install -y git python3
```

```
echo -e 'export HY_APP_PLATFORMS="linux/arm64,windows/arm64"' >> /etc/environment
export HY_APP_PLATFORMS="linux/arm64,windows/arm64"
echo $HY_APP_PLATFORMS
```

首次编译

```
git clone https://github.com/apernet/hysteria.git
cd hysteria
python3 ./hyperbole.py build
cd ..
```

再次编译

```
cd hysteria
git pull
python3 ./hyperbole.py build
cd ..
```

复制文件

**linux-arm64**

```
cp -f hysteria/build/hysteria-linux-arm64 /usr/local/bin/hysteria
chmod +x /usr/local/bin/hysteria
```

**windows-amd64**

```
cp -f hysteria/build/hysteria-windows-arm64.exe ./hysteria.exe
```
